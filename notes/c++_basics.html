C++ BASICS

HelloWorld Program - structure

            
                #include <iostream>
                using namespace std;
                
                int main()
                {
                    std::cout << "Hello World!" << std::endl;
                    return 0;
                }

The elements in the Hello World Program:

Element Function
#include Preprocessor directive; includes the required header files into the program
<iostream> iostream.h, the header file containing standard input and output operations in C++
int Integer datatype
main() The main function, code execution starts here
std The namespace, it is like a library containing some builtin functions
:: The scope resolution operator, mentions the scope of the function or name of its namespace
cout The output function, prints data to stdout
endl The endline function, moves the cursor to a new line
